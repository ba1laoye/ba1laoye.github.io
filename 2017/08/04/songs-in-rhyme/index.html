歌曲押韵识别

发表于 2017-08-04

背景

今天早上听歌的时候，发现押韵的歌曲很有意思。
于是想把所有押韵的歌曲找出来欣赏一遍。
但现在似乎，好像也没有一个检索押韵的功能，那我就想自己来做一个。
（网易云音乐上有歌单，但不全，而且对于押什么韵也没有分析）

思路

1. 肯定是获取歌曲、歌词数据，这块计划从网易云、虾米、豆瓣上抓取歌词；
2. 其次是研究中文、英文押韵规则；
3. 再才是做押韵的模型：
    * 歌词分句
    * 句尾截取
    * 原音提取
    * 韵律分析
4. 最后做一个输出界面；

预期的产出

输入：一个歌曲名称；
输出：是否押韵，什么形式的押韵；

难点

中文押韵规则和英文押韵规则不一样，需要区分处理；

脑洞

*. 日文、泰文、西语、法语是否也有押韵规则？
*. 会否有中英文混合的押韵情况？
